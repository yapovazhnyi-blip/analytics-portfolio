# analytics-portfolio
End-to-end data analytics portfolio using BigQuery, Google Sheets, and Tableau. Demonstrates SQL data cleaning, joins, aggregations, and interactive dashboard creation with business insights.

This repository showcases an end-to-end data analytics project built with BigQuery, Google Sheets, and Tableau.

The project demonstrates practical data analyst skills including:
- Data cleaning and transformation using SQL
- Table joins, aggregations, and analytical queries
- Creation of temporary and reporting tables
- Exploratory analysis in Google Sheets
- Interactive dashboard design in Tableau Public

The focus is on transforming raw transactional data into clear business insights suitable for decision-making.
